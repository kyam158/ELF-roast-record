# ELF Roast Record

焙煎士が毎日使う記録帳を想定した、軽量なコーヒー焙煎記録Webアプリです。

## 構成

- `index.html` - 画面構造
- `style.css` - レイアウトと見た目
- `script.js` - 計算、保存、読込、削除
- `logo.png` - ELFロゴ

## 機能

- 基本情報、生豆情報、投入条件の記録
- 重量減少率の自動計算
- イベント入力とフェーズ時間、割合の自動計算
- 0:00から15:00までの固定ログ表
- メモ入力
- localStorageによるオートセーブ
- 履歴保存、復元、削除

## 使い方

`index.html` をブラウザで開いて使用します。外部ライブラリは使用していません。
